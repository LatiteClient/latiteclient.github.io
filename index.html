Frequently Asked Questions

What is Latite Client?
Latite Client is a brand new Minecraft: Windows 10 Edition legit PVP-oriented DLL client!
What features does Latite Client provide?
Latite Client provides many mods. Features include:
  • Armor HUD
  • BehindYou (Snaplook)
  • Bow Indicator
  • Break Indicator
  • Block Outline
  • CPS Counter
  • FPS Display
  • Freelook
  • Fullbright
  • Item Render
  • Clock
  • Position Display (displays your coordinates)
  • Keystrokes
  • Hitboxes
  • Server Display (displays the IP address of the server you're on)
  • Smooth Camera (cinematic camera)
  • Toggle Sprint
  • Zoom
  • Third person nametag
  • Movable Scoreboard
  • Movable Paperdoll
  • Movable Bossbar
  • Bottom Chat
How do I open the Mod Menu?
To open the mod menu, you can either click 'M' or the 'INS' key on your keyboard.
How do I edit the module positions?
Click 'I' on your keyboard. You can change this in the settings menu of the UI.
What versions of Minecraft does Latite Client support?
Latite currently supports:
  • • 1.20 - 1.20.1
  • • 1.18.10 - 1.18.12
What device does Latite Client support?
Latite Client only supports 64 bit system with a 64 bit build of Minecraft on Windows 10 or 11. You cannot use Latite with 32 bit builds of Minecraft on a 64 bit machine. You also cannot use Latite on a 32 bit machine. If you try for yourself, it just wouldn't work.
